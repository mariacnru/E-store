# 🛒 E-Store

A modern and responsive e-commerce web application built with **React**, **Vite**, and **Tailwind CSS**. The project focuses on clean UI, performance optimization, accessibility, and a smooth user experience across all devices.

![Preview](./public/preview.webp)

---

## ✨ Features

- 📦 Product listing and category pages
- 🔍 Product filtering by category
- 🛍️ Product details page with image gallery
- ❤️ Wishlist functionality using Context API
- 🛒 Shopping cart management
- 📄 Pagination for product lists
- 📞 Contact page with form validation
- ❓ FAQ section with accordion interface
- 📱 Fully responsive design
- ⚡ Optimized images using WebP format
- ♿ Improved accessibility and SEO
- 🚀 Fast development environment with Vite

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/USERNAME/e-store.git
```

### Navigate to the project folder

```bash
cd e-store
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```
